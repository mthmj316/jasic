Ein Bezugssystem kann als orts- und zeitabhängiger Punkt gesehen werden, von dem aus ein bestimmter Vorgang beobachtet wird.

Je nachdem von welchem Bezugssystem (Beobachtungspunkt) ein Vorgang beobachtet wird, können Messungen unterschiedliche Ergebnisse liefern.

Beispiel für unterschiedliche Bezugssysteme:

Wenn sich eine Person in einem fahrenden Zug (Geschwindigkeit des Zugs = \(100\,{{km}\over{h}}\)) mit einer Geschwindigkeit von \(3\,{{km}\over{h}}\) bewegt, dann ist die beobachtete Geschwindigkeit der Person für jemanden im Zug niedriger als für jemanden im Bahnhof, der beobachtet, wie der Zug vorbeifährt.

In diesem Fall gibt es also zwei Bezugssysteme (Beobachtungspunkte):

  1. innerhalb des Zuges (\(v_{Person} = 3\,{{km}\over{h}} \)) und
  2. im Bahnhof \((v_{Person} = 100\,{{km}\over{h}} + 3\,{{km}\over{h}} = 103\,{{km}\over{h}})\)
